# 🌊 Vida Abissal

Um site interativo onde você mergulha pelo oceano simplesmente rolando a página.
A ideia do projeto foi criar uma experiência diferente: em vez de só mostrar informações, você literalmente “desce” pelo oceano e vai descobrindo as criaturas conforme a profundidade aumenta.
Acesse o projeto
🔗 https://vida-abissal.vercel.app
## 🧠 Como funciona?
Conforme você rola a página, a profundidade em metros aumenta e novos animais aparecem exatamente nas regiões onde vivem na vida real.
Cada criatura traz:
- uma descrição mais completa  
- uma curiosidade interessante  
- a zona do oceano onde ela pertence  
A intenção foi deixar tudo com uma pegada meio documental, como se fosse uma exploração guiada.
## 🌊 O que você vai encontrar
Durante a descida, você passa por diferentes zonas do oceano:
- Zona de Luz Solar  
- Zona do Crepúsculo  
- Zona da Meia-Noite  
- Zona Abissal  
- Zona Hadal  
E no final… você chega ao limite conhecido da vida no oceano.
## ✨ O que esse projeto tem
- Scroll que simula profundidade real  
- Animações suaves conforme os elementos aparecem  
- Sidebar interativa com informações  
- Efeito de água usando shader (Three.js)  
- Escurecimento progressivo conforme você desce  
- Evento final quando chega nas maiores profundidades

## 💡 Por que eu fiz isso?
Sempre achei o oceano profundo uma das coisas mais interessantes (e assustadoras) que existem.
Esse projeto foi uma forma de juntar programação com essa curiosidade, criando algo visual e interativo em vez de só texto.
